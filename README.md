# Global Startup Investment & Success Analysis

## 📌 Project Overview
ניתוח מקיף של נתוני השקעות וסיכויי הצלחה של סטארט-אפים בעולם. 
הפרויקט כולל ניקוי נתונים מתקדם ב-SQL וויזואליזציה אינטראקטיבית ב-Tableau.

## 🛠 Tech Stack
- **SQL (PostgreSQL):** ניקוי נתונים, עבודה עם Window Functions וחישובי Aggregations.
- **Tableau:** בניית דאשבורד אינטראקטיבי וניתוח גאוגרפי.
